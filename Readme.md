# AI PDF Chatbot

An AI-powered chatbot that allows users to upload PDFs and ask questions about the document.

## Features
- PDF Upload
- AI Question Answering
- Document Summarization
- Semantic Search
- LangChain + Groq Integration

## Tech Stack
- Python
- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Groq LLM
